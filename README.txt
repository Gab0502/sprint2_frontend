*------------------------------------------------------------------*
Descrição
*------------------------------------------------------------------*
Nosso projeto é um site baseado no problema de enchentes em São Paulo. Ele irá, por meio de sensores localizados nos esgotos, avisar a população e responsáveis previamente sobre possíveis enchentes em ruas do estado, ajudando moradores a se prepararem diminuindo ou evitando os danos causados pelo desastre.

*------------------------------------------------------------------*
Header
*------------------------------------------------------------------*
O header do site, apresentado em cima de todas as páginas, possui um menu sanduíche que será ativado ao passar o mouse por cima de seu ícone, apresentando links para todas as paginas do site. Ao lado do menu temos o ícone do site, que funcionará como botão HOME. No lado direito do header, temos o botão que levará à pagina de login e mais dois botões que funcionam como o darkmode do site.


*------------------------------------------------------------------*
HOME
*------------------------------------------------------------------*
Página inicial do projeto, possui um slider que mostra imagens sobre o projeto e abaixo há informações sobre enchentes.


*------------------------------------------------------------------*
Sobre Nós
*------------------------------------------------------------------*
Página contando sobre como foi o início do trabalho e como foi iniciada a idealização dele.


*------------------------------------------------------------------*
Software
*------------------------------------------------------------------*
Detalhamento sobre como funciona o software do site.


*------------------------------------------------------------------*
Hardware
*------------------------------------------------------------------*
Detalhamento sobre os sensores utilizados no projeto, conta com imagens ilustrativas, assim como vídeo explicativo sobre como o projeto funciona.


*------------------------------------------------------------------*
Contato
*------------------------------------------------------------------*
Possui informações para o contato da equipe de administradores do site e também um formulário onde é possível enviar um e-mail diretamente para a equipe, caso o usuário possua algum aplicativo de email no seu computador.


*------------------------------------------------------------------*
Login
*------------------------------------------------------------------*
Página utilizada para o login dos usuários do site. Atualmente possuímos o login "morador", que irá para a página mostrada para moradores e o login "adm" que irá levar para a pagina de administradores.


*------------------------------------------------------------------*
Login morador
*------------------------------------------------------------------*
Página que moradores utilizarão, pode-se pesquisar o nome das ruas sugeridas na página para puxar diferentes resultados.


*------------------------------------------------------------------*
Login adm
*------------------------------------------------------------------*
Página que administradores utilizarão. Ela exibirá um mapa com algumas ruas, clicando nos pontos no mapa, eles levarão para páginas com detalhes sobre a região selecionada.
